## What is Hermes Agent?

Hermes Agent is an intelligent assistant that lives in your terminal (and desktop). It can:

- **Write code** in any language and push to GitHub
- **Manage repositories** — create, fork, clone, configure
- **Run CI/CD** — trigger workflows, manage secrets, create releases
- **Research** — search arXiv, browse the web, extract YouTube transcripts
- **Generate media** — images, audio, diagrams, ASCII art
- **Automate** — cron jobs, scheduled tasks, background processes

## Getting Started

```bash
# Install via pip
pip install hermes-agent

# Or run directly
hermes
```
